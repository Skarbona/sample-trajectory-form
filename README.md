# Trajectory Form

Prosty formularz w **React** (CDN) + **Tailwind CSS**.

## Demo

https://skarbona.github.io/sample-trajectory-form/

## Lokalnie

```bash
python3 -m http.server 8080
```

Otwórz `http://localhost:8080`.

## Zachowanie

### Nawigacja

- **Option One** / **Option Two**

### Option One

- Textarea **All Issues** z przykładową treścią
- Slider **Trajectory Quality** (1–5)

### Option Two

- Slider **Trajectory Quality** (1–5)
- Bloki **Issue 1**, **Issue 2**, … (textarea + Severity: Low / Medium / Hard)
- Przycisk **Add new Issue**
